* 不出意外，固件每周五上午更新！

* X86_64全能版各种插件都内置了，不带istore

* X86_64精简版仅带iStore，无Dcoker，极速轻量

* X86_64-R2203精简版带QuickStart便捷首页和iStore，有Docker，基于官方openwrt-r2203分支

* AX6/AX3600/R2S/R4s仅带iStore
